# TODO — פיצ'ר ניתוח ויזואלי (Vision AI) ל-BrikOps

## סטטוס: לבחינה עתידית

## הרעיון
שילוב Vision API (Claude / OpenAI) לניתוח תמונות ליקויים, QC, ומסירות.

## 6 יישומים אפשריים

### 1. זיהוי אוטומטי של סוג ליקוי (הכי שימושי)
PM מצלם → AI ממלא: קטגוריה, כותרת, חומרה.
חיסכון: ~30 שניות × 500 ליקויים = 4 שעות.

### 2. זיהוי חומרה (severity)
AI מדרג: קוסמטי / מבני / בטיחותי.
סדק בטיח = קוסמטי. סדק בקורת בטון = מבני → חוסם מסירה.

### 3. QC אוטומטי
מצלם קיר/ריצוף → AI בודק: ישר? אחיד? פגמים?

### 4. השוואת before/after
תמונת ליקוי מקורית ← תמונת תיקון → AI משווה ומאשר/דוחה.

### 5. קריאת מונים (handover)
מצלם מונה מים/חשמל → AI קורא מספר → ממלא שדה אוטומטית.

### 6. בדיקת שלמות (completeness check)
מצלם חדר → AI מזהה: חסר שקע, ברז לא מותקן, מזגן לא מחובר → יוצר ליקויים.

## ארכיטקטורה מוצעת
```
PM מצלם ליקוי
    ↓
Frontend שולח תמונה ל-backend
    ↓
Backend שולח ל-Claude Vision API:
  "נתח את תמונת הליקוי הזו.
   החזר JSON: {category, title, severity, description}"
    ↓
Backend ממלא שדות אוטומטית
    ↓
PM רואה טופס מלא — רק מאשר/מתקן
```

## עלות
- Claude Vision: ~$0.01-0.03 לתמונה
- 500 ליקויים = $5-15
- זניח ביחס לערך
