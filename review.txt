NEW  frontend/src/services/offlineCache.js
-------------------------------------------------------------------------------
Dependency-free IndexedDB wrapper. DB "brikops-offline", store "reads"
(keyPath: url, index ts). Fail-soft: every fn wrapped so a missing/blocked
IndexedDB (private mode, old WebView) NEVER throws → no-op/null. Records
{ url, data, ts }. Surface (exact contract):
   cachePutRead(url, data)  — fire-and-forget write; never throws.
                              opportunistic cacheEvict every 25th write (bounded).
   cacheGetRead(url)        — returns stored data or null; never throws.
   cacheEvict({maxAgeMs=7d, maxEntries=400}) — drop stale + cap newest.
   cacheClearAll()          — wipe; never throws.
   _openDb() memoised; onerror/onblocked/try-catch all resolve(null).
   NO PII logging, NO console.log of bodies, NO @capacitor import.
