WHAT
----
Per Zahi staging smoke 2026-05-04 after #486 deploy: project's
buildings are named "בניין A" and "בניין B" — both produce the
SAME single-letter badge "ב" because sort_index is unset on these
docs and fallback uses first letter. User can't tell the two
buildings apart visually. Replace single-letter circle badge with
full building name in a violet pill (truncate + tooltip for very
long names). Widen "בניין" sticky column 80px → 120px to fit.
