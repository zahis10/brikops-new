# App Privacy Practices Questionnaire

**חובה למלא ב-App Store Connect → App Privacy.** להלן התשובות המדויקות לשאלון של Apple:

### Do you collect data from this app?
**Yes**

### Data Types Collected

**Contact Info:**
- Email Address → Data Linked to User → Used for App Functionality (Account auth via Apple Sign-In)
- Name → Data Linked to User → Used for App Functionality (User profile display)
- Phone Number → Data Linked to User → Used for App Functionality (אימות משתמש / OTP / התראות WhatsApp+SMS על עדכוני משימות)

**Identifiers:**
- User ID → Data Linked to User → Used for App Functionality (account identification)

**Usage Data:**
- Product Interaction → Not Linked to User → Used for Analytics / App Functionality (פעולות בתוך האפליקציה למעקב אחר תהליך הליקויים)

**User Content:**
- Photos or Videos → Data Linked to User → Used for App Functionality (צילום ליקויים באתר)
- Other User Content → Data Linked to User → Used for App Functionality (תיאורי ליקויים, מיקום בדירה, הערות)

### Data Used to Track You?
**No** — BrikOps לא מעבירה ולא משתפת נתונים עם third parties למטרות tracking או advertising.

# Export Compliance (ITSAppUsesNonExemptEncryption)

**Answer:** **No** — האפליקציה משתמשת רק ב-HTTPS/TLS סטנדרטי של iOS. אין encryption custom.
