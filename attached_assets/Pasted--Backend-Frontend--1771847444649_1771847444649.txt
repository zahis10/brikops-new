A) FIX קריטי – Hierarchy נטען אבל הפרונט מפרסר שדה לא נכון
✅ ה-endpoint מחזיר:
{ project_id, project_name, buildings: [...] }
❌ בפרונט כרגע:
const buildings = Array.isArray(data) ? data : (data?.items || []);
✅ לתקן לפרסור נכון וחסין:
const buildings =
  Array.isArray(data) ? data :
  Array.isArray(data?.buildings) ? data.buildings :
  Array.isArray(data?.items) ? data.items :
  [];

וגם:
- empty state (“צריך להקים מבנה וקומות”) יוצג רק אם buildings.length === 0 אחרי הפרסור.
- להוסיף console.warn פעם אחת אם data קיים אבל buildings ריק: keys של response.

B) FIX קריטי – service לא נכון (TypeError) + import נכון
בדף QC floor selection להפסיק לקרוא:
buildingService.getHierarchy(projectId)
ולשנות ל:
projectService.getHierarchy(projectId)
ולוודא שה-importים תואמים.
