# #386 — Phase 7 Wave 1 — Splash Experience (D2 Daily + D3-A First-Launch)

## What & Why
החליפו את ה-LoadingSpinner וה-ConnectingScreen הבסיסיים ב-App.js במסך טעינה מותג-אמיתי
לאפליקציית BrikOps. המסך הזה משתמש בשני וריאנטים:
- **D2 (Skeleton + Shimmer)** — טעינה יומיומית. מהיר, רואים סקלטון דומה למסך האמיתי,
  shimmer RTL שעובר פעם אחת. זה המסך שכל משתמש רואה 99% מהפתיחות.
- **D3-A (Brick Assembly)** — אנימציה של לבנים שנופלות ונבנות לאות B. מוצג רק
  בהתקנה ראשונה ואחרי עדכון גרסה. פעם אחת → אף פעם לא שוב עד גרסה חדשה.

ההחלטה בין שני הוריאנטים נקבעת דרך `@capacitor/preferences` שמשווה בין הגרסה
הנוכחית של האפליקציה לגרסה האחרונה שהמשתמש ראה.

הקבצים מקור נמצאים ב-`design/` בשורש הריפו (ייבוא ישיר מ-Claude Design).

## Done looks like
- ✅ משתמש שמתקין את האפליקציה לראשונה → רואה את אנימציית הלבנים (D3-A) פעם אחת,
  אחרי זה המערכת שומרת את הגרסה ופותחת את הדשבורד.
- ✅ משתמש שפותח את האפליקציה ביום הבא (אותה גרסה) → רואה את D2 (Skeleton + Shimmer),
  לא את אנימציית הלבנים. ההבדל בולט — אין הצגה, ישר סקלטון מודרני.
- ✅ משתמש שמעדכן ל-1.0.11 (מ-1.0.10) → רואה את D3-A שוב פעם אחת.
- ✅ אין LoadingSpinner הישן (הספינר הקטן על רקע slate). הוחלף בכל מקום.
- ✅ אין `ConnectingScreen` הישן עם "מתחבר..." על רקע slate-50. הוחלף ב-D2 עם
  הטקסט "מתחבר..." כחלק מהסקלטון.
- ✅ D2 מסתיים ברגע שהדאטה מגיעה (לא טיימר קבוע). אם הדאטה מגיעה ב-300ms,
  D2 נעלם ב-300ms + 500ms fade = 800ms סה"כ.
- ✅ D3-A רץ לכל היותר 1600ms (duration של animation) + 500ms fade, ואז דשבורד.
- ✅ RTL נכון — ה-shimmer זורם מימין לשמאל.
- ✅ בלי console errors.
- ✅ בלי שכבות לבנות שנראות רגע לפני הסקלטון (דפוס קלאסי של flash of unstyled).

- ❌ לא להוסיף Skeleton componentים חדשים לדפים פנימיים — הסקלטון בסקלטון loading

**File: `frontend/src/components/splash/BrikSplash.jsx`**

זה ה-wrapper הראשי שמחליט מה להציג. צור אותו חדש:

```jsx
import React, { useEffect, useState } from 'react';
import { Preferences } from '@capacitor/preferences';
import { App } from '@capacitor/app';
import D2DailySplash from './D2DailySplash';
import D3ABrickAssembly from './D3ABrickAssembly';

const LAST_SEEN_VERSION_KEY = 'brikops_last_seen_version';

export default function BrikSplash({ isReady = false, loadingText = 'מתחבר' }) {
  const [variant, setVariant] = useState(null); // null → D2 → D3A
  const [d3aDone, setD3aDone] = useState(false);

  useEffect(() => {
    (async () => {
      try {
        const { version: currentVersion } = await App.getInfo();
        const { value: lastSeenVersion } = await Preferences.get({
          key: LAST_SEEN_VERSION_KEY,
        });

        if (!lastSeenVersion || lastSeenVersion !== currentVersion) {
          // התקנה ראשונה או אחרי עדכון
          setVariant('D3A');
          await Preferences.set({
            key: LAST_SEEN_VERSION_KEY,
            value: currentVersion,
          });
        } else {
          setVariant('D2');
        }
      } catch (e) {
        // אם App.getInfo נכשל (למשל ב-dev ב-web) → fallback ל-D2
        console.warn('[BrikSplash] version check failed, using D2:', e);
        setVariant('D2');
      }
    })();
  }, []);

  if (variant === null) {
    // 50-100ms של null עד שה-Preferences עונה. אל תציג כלום — לא גם רקע לבן.
    // הרקע הזה תואם את native splash של iOS/Android ולא יוצר הבהוב.
    return (
      <div style={{
        position: 'fixed', inset: 0,
        background: 'linear-gradient(180deg, #3A4258 0%, #323A4E 50%, #2A3142 100%)',
      }}/>
    );
  }

  if (variant === 'D3A' && !d3aDone) {
    return <D3ABrickAssembly onComplete={() => setD3aDone(true)} />;
  }

  // D2 או D3A שהסתיים → fallback ל-D2 שמחכה ל-isReady
  return <D2DailySplash isReady={isReady} loadingText={loadingText} />;
}
```
