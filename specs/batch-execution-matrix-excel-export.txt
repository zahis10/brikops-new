================================================================================
DESIGN — what gets exported, where, and how
================================================================================

WHAT GETS EXPORTED:
  - The CURRENT FILTERED VIEW from the user's session — frontend sends
    explicit `unit_ids` + `stage_ids` lists derived from useMatrixFilters
    + the visible stages array. Backend exports exactly what's listed.
  - Backend validates ownership: `unit_ids` MUST belong to `project_id`,
    `stage_ids` MUST be in the project's matrix (visible OR hidden — for
    flexibility, since admin might want to export hidden stages too).
  - If frontend sends empty `unit_ids` → export ALL units.
  - If frontend sends empty `stage_ids` → export ALL visible stages.

LAYOUT:
  - Single sheet "מטריצת ביצוע"
  - Columns: בניין | קומה | דירה | מס׳ חדרים? | <stages in matrix order>
    (the location columns mirror what the matrix already shows; custom
    stages render before template stages — same negative-order trick)
  - Header row frozen
  - First 3 cols frozen (so scrolling right keeps building/floor/unit visible)
  - Auto-width per column (clamp at 30 chars)
  - RTL view: `ws.sheet_view.rightToLeft = True`

CELL FORMATTING:
  - Status cells: status label in Hebrew (e.g. "הושלם", "לא בוצע")
    + light cell color matching MATRIX_STATUSES ramp:
        completed     → light green   (#D1FAE5)
        partial       → light amber   (#FEF3C7)
        in_progress   → light blue    (#DBEAFE)
        not_done      → light red     (#FEE2E2)
        not_relevant  → light slate   (#F1F5F9)
        no_findings   → light teal    (#CCFBF1)
        empty         → no fill
  - Tag cells: text_value as plain text. Empty = blank.
  - Notes: ATTACHED AS EXCEL COMMENTS on the cell (popup on hover).
    Cleaner than inline suffix, Excel-native UX.

FILENAME:
  - `מטריצת ביצוע - {project_name} - {YYYY-MM-DD}.xlsx`
  - Use RFC 5987 `Content-Disposition: attachment; filename*=UTF-8''<encoded>`
    so Hebrew filenames work in Chrome / Safari / Firefox / iOS WebView.
  - ASCII fallback `filename="execution-matrix-{date}.xlsx"` for old clients.

DOWNLOAD ON CLIENT:
  - Use `downloadBlob()` from `frontend/src/utils/fileDownload.js`
    (per CLAUDE.md Capacitor rule — `<a download>` won't work on native).
  - On native: filesystem write + Share sheet; on web: blob URL trigger.
  - Mime type: `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`

AUTHORIZATION:
  - Re-uses `_check_matrix_view` (super_admin / project_manager / owner /
    management_team). Contractor blocked.
  - 403 if user lacks view access on the project.

Implementation outline:
    from io import BytesIO
    from openpyxl import Workbook
    from openpyxl.styles import Alignment, Font, PatternFill
    from openpyxl.comments import Comment
    from openpyxl.utils import get_column_letter

    STATUS_LABELS = {
        "completed":     "הושלם",
        "partial":       "בוצע חלקית",
        "in_progress":   "בתהליך",
        "not_done":      "לא בוצע",
        "not_relevant":  "לא רלוונטי",
        "no_findings":   "אין חוסרים",
    }

    STATUS_FILLS = {
        "completed":     PatternFill("solid", fgColor="D1FAE5"),  # green-100
        "partial":       PatternFill("solid", fgColor="FEF3C7"),  # amber-100
        "in_progress":   PatternFill("solid", fgColor="DBEAFE"),  # blue-100
        "not_done":      PatternFill("solid", fgColor="FEE2E2"),  # red-100
        "not_relevant":  PatternFill("solid", fgColor="F1F5F9"),  # slate-100
        "no_findings":   PatternFill("solid", fgColor="CCFBF1"),  # teal-100
    }

    def build_matrix_xlsx(project, units, stages, cells, buildings, floors):
        wb = Workbook()
        ws = wb.active
        ws.title = "מטריצת ביצוע"
        ws.sheet_view.rightToLeft = True

        # Header row
        headers = ["בניין", "קומה", "דירה", "מס׳ חדרים"]
        for s in stages:
            headers.append(s.get("title", "—"))
        ws.append(headers)
        for col_idx in range(1, len(headers) + 1):
            cell = ws.cell(row=1, column=col_idx)
            cell.font = Font(bold=True, size=11)
            cell.alignment = Alignment(horizontal="right", vertical="center", wrap_text=True)
            cell.fill = PatternFill("solid", fgColor="F3F4F6")  # slate-100

        # Data rows
        cells_index = {(c["unit_id"], c["stage_id"]): c for c in cells}
        for row_idx, unit in enumerate(units, start=2):
            building = buildings.get(unit.get("building_id"), {})
            floor = floors.get(unit.get("floor_id"), {})

            ws.cell(row=row_idx, column=1, value=building.get("name", "—"))
            ws.cell(row=row_idx, column=2, value=floor.get("floor_number", "—"))
            ws.cell(row=row_idx, column=3, value=unit.get("unit_no", "—"))
            ws.cell(row=row_idx, column=4, value=unit.get("room_count") or "")

            for stage_idx, stage in enumerate(stages):
                col = 5 + stage_idx
                cell_xl = ws.cell(row=row_idx, column=col)
                c = cells_index.get((unit["id"], stage["id"]))
                if not c:
                    continue
                if stage.get("type") == "tag":
                    cell_xl.value = c.get("text_value") or ""
                else:
                    status = c.get("status")
                    if status:
                        cell_xl.value = STATUS_LABELS.get(status, status)
                        cell_xl.fill = STATUS_FILLS.get(status, PatternFill())
                # Note as Excel comment (truncate at 500 to be safe)
                # Author = whoever last edited the cell (for audit-trail
                # context inside Excel — PM hovering sees the name).
                # Cells MUST be enriched via _summarize_cell BEFORE
                # being passed in (see PART 2 endpoint — uses summary
                # form so last_actor_name is populated).
                note_raw = c.get("note") or ""
                note_clean = note_raw.strip()
                if note_clean:
                    author = c.get("last_actor_name") or "BrikOps"
                    cell_xl.comment = Comment(note_clean[:500], author)

        # Frozen panes — first 3 cols + header row
        ws.freeze_panes = "E2"

        # Auto-width (clamp 8..30 chars)
        for col_idx, header in enumerate(headers, start=1):
            col_letter = get_column_letter(col_idx)
            max_len = max(
                [len(str(header))]
                + [len(str(ws.cell(row=r, column=col_idx).value or ""))
                   for r in range(2, ws.max_row + 1)]
            )
            ws.column_dimensions[col_letter].width = max(8, min(30, max_len + 2))

        buf = BytesIO()
        wb.save(buf)
        buf.seek(0)
        return buf

    from contractor_ops.execution_matrix_export import build_matrix_xlsx
    from fastapi.responses import StreamingResponse
    from urllib.parse import quote

    @router.post("/{project_id}/export.xlsx")
    async def export_matrix_xlsx(
        project_id: str,
        payload: dict,  # {unit_ids?: [], stage_ids?: []}
        user: dict = Depends(get_current_user),
    ):
        """Export current matrix view as .xlsx. Frontend sends explicit
        unit_ids + stage_ids lists derived from active filters. Empty
        lists = export all visible.
        """
        db = get_db()
        await _check_matrix_view(user, project_id)  # 403 if not authorized

        project = await db.projects.find_one({"id": project_id}, {"_id": 0})
        if not project:
            raise HTTPException(404, "Project not found")

        # Validate scope: requested IDs must belong to this project
        requested_unit_ids = payload.get("unit_ids") or []
        requested_stage_ids = payload.get("stage_ids") or []

        unit_query = {"project_id": project_id, "archived": {"$ne": True}}
        if requested_unit_ids:
            unit_query["id"] = {"$in": requested_unit_ids}

        # Hard cap of 2000 units per export — fast-fail rather than
        # silent truncation by `to_list(2000)`. Real-world projects
        # with 4 buildings × 50 floors × ~10 units/floor = 2000+ are
        # plausible; PM should narrow via filters before exporting.
        total_count = await db.units.count_documents(unit_query)
        if total_count > 2000:
            raise HTTPException(
                400,
                "ייצוא של יותר מ-2000 דירות אינו נתמך. סנן את התצוגה לפני הייצוא.",
            )

        units = await db.units.find(unit_query, {"_id": 0}).to_list(2000)

        # Re-use existing helpers from #483 to resolve stages
        matrix_config = await _get_or_init_matrix_config(db, project_id)
        tpl = await _get_template(db, project_id=project_id)
        all_stages = _resolve_visible_stages(matrix_config, tpl)
        if requested_stage_ids:
            stages = [s for s in all_stages if s["id"] in requested_stage_ids]
        else:
            stages = all_stages

        # Fetch buildings + floors + cells
        building_ids = list({u["building_id"] for u in units if u.get("building_id")})
        floor_ids = list({u["floor_id"] for u in units if u.get("floor_id")})
        buildings = {
            b["id"]: b for b in await db.buildings.find(
                {"id": {"$in": building_ids}}, {"_id": 0}
            ).to_list(100)
        }
        floors = {
            f["id"]: f for f in await db.floors.find(
                {"id": {"$in": floor_ids}}, {"_id": 0}
            ).to_list(500)
        }
        unit_id_set = {u["id"] for u in units}
        stage_id_set = {s["id"] for s in stages}
        cells = await db.execution_matrix_cells.find(
            {
                "unit_id": {"$in": list(unit_id_set)},
                "stage_id": {"$in": list(stage_id_set)},
            },
            {"_id": 0},
        ).to_list(50000)

        # Enrich cells via _summarize_cell so last_actor_name is
        # populated — needed by build_matrix_xlsx for Excel comment
        # author. Same shape as /matrix endpoint cells.
        cells_summary = [_summarize_cell(c) for c in cells]

        # Build xlsx
        buf = build_matrix_xlsx(project, units, stages, cells_summary, buildings, floors)

        # Filename with RFC 5987 encoding for Hebrew
        # Sanitize project name — strip filesystem-illegal chars that
        # break filenames on Windows / iOS / macOS:
        #   /  \  :  *  ?  "  <  >  |
        # (Hebrew chars + spaces + parentheses are all fine.)
        import re
        from datetime import datetime
        date_str = datetime.utcnow().strftime("%Y-%m-%d")
        project_name_raw = project.get("name", "matrix")
        project_name = re.sub(r'[/\\:*?"<>|]', '_', project_name_raw).strip() or "matrix"
        filename_he = f"מטריצת ביצוע - {project_name} - {date_str}.xlsx"
        filename_ascii = f"execution-matrix-{date_str}.xlsx"
        encoded = quote(filename_he, safe="")
        disposition = (
            f'attachment; filename="{filename_ascii}"; '
            f"filename*=UTF-8''{encoded}"
        )

        return StreamingResponse(
            buf,
            media_type=(
                "application/vnd.openxmlformats-officedocument."
                "spreadsheetml.sheet"
            ),
            headers={"Content-Disposition": disposition},
        )

    async exportXlsx(projectId, { unit_ids = [], stage_ids = [] } = {}) {
      const url = `${API}/execution-matrix/${projectId}/export.xlsx`;
      const response = await axios.post(
        url,
        { unit_ids, stage_ids },
        {
          headers: getAuthHeader(),
          responseType: "blob",
        }
      );
      // Server returned a binary xlsx — pull suggested filename from header
      const cd = response.headers["content-disposition"] || "";
      let filename = `execution-matrix-${new Date().toISOString().slice(0,10)}.xlsx`;
      const match = cd.match(/filename\*=UTF-8''([^;]+)/);
      if (match) {
        try { filename = decodeURIComponent(match[1]); } catch {}
      } else {
        const ascii = cd.match(/filename="([^"]+)"/);
        if (ascii) filename = ascii[1];
      }
      return { blob: response.data, filename };
    },
