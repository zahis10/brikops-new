================================================================================
CONTEXT — what this feature is, why it exists
================================================================================

PMs and execution engineers in production today maintain external Google
Sheets to track project execution at a glance. Eli's tracker observed
2026-05-04 (~/Downloads/טבלת מעקב בניין 8+9 -אלי - Google Sheets.pdf):
50 units × 20 stages, simple status values per cell, free-text notes,
sales-status columns, lobby/common-area rows.

The gap: BrikOps's existing בקרת ביצוע is documentation-heavy (photos,
notes, signatures) — too slow for "where's everything at?" management
overview. Engineers fall back to external Excel.

Goal: replace the external Excel WITHIN BrikOps. Not a fancy dashboard
— literally an Excel-replacement experience tightly integrated with
project data, with light read-only annotations from the formal record.

Architectural decision (Zahi 2026-05-04):
  - Matrix = INDEPENDENT data store (own collections, own state)
  - Annotations from qc_runs are READ-ONLY display hints (no state coupling)
  - Marking a cell in the matrix does NOT modify qc_runs
  - Admin edits to qc_template auto-reflect in matrix base stages
    (computed at render time, no migration)
  - Engineers can ADD custom stages + REMOVE base stages (per-project,
    not per-user)

Out of scope for PHASE 1 (deferred):
  - Frontend (PHASE 2)
  - Read-only 📎 annotation overlay (PHASE 2 frontend OR Tier 2)
  - Excel/PDF export (Tier 2)
  - Realtime collaboration (Tier 3, requires WebSocket)
  - Per-user matrix variants (intentionally rejected — one matrix per project)
  - Shared saved views across project members (currently per-user only;
    add `is_shared` flag in PHASE 2 if requested)
  - Stage reordering endpoint (PHASE 2 — currently order is set on create
    and changed only via full custom_stages_added replace)
  - Cleanup of orphaned cells when custom stage deleted (PHASE 2 —
    currently cells with hidden stages are simply not rendered)
  - GDPR-compliant audit anonymization on user account deletion (Tier 2 —
    currently audit retains actor_id forever; requires sweep across all
    audit-bearing collections, not matrix-specific)
  - Pagination for large projects (>100 units / >3MB response) — current
    response returns all cells in one query; PHASE 2 frontend may
    paginate client-side first, backend pagination if scale demands

       MATRIX_STATUS_VALUES = (
           'completed',     # ✓ בוצע
           'partial',       # ⚠ חלקי
           'in_progress',   # ◷ בעבודה
           'not_done',      # ✗ לא בוצע
           'not_relevant',  # − אין צורך
           'no_findings',   # ⊘ אין חוסרים
       )

       MATRIX_STAGE_TYPES = ('status', 'tag')

       class MatrixStageCreate(BaseModel):
           """Custom stage added by approver/PM (NOT part of qc_template)."""
           title: str = Field(..., min_length=1, max_length=80)
           type: Literal['status', 'tag'] = 'status'
           order: Optional[int] = None  # if None, append to end

       class MatrixStage(BaseModel):
           """Stage entry in the matrix — either base (from qc_template) or custom."""
           id: str
           title: str
           type: Literal['status', 'tag']
           order: int
           source: Literal['base', 'custom']  # base = from qc_template; custom = engineer-added
           # Base stages additionally include:
           scope: Optional[Literal['floor', 'unit']] = None  # only for base stages

       class MatrixStagesUpdate(BaseModel):
           """PATCH-style update for the project matrix's stage list."""
           custom_stages_added: Optional[List[MatrixStageCreate]] = None  # full replace if provided
           base_stages_removed: Optional[List[str]] = None  # stage_ids hidden from base
           # If both omitted → no change.

       class MatrixCellAuditEntry(BaseModel):
           actor_id: str
           actor_name: str
           timestamp: str  # ISO
           status_before: Optional[str] = None
           status_after: Optional[str] = None
           note_before: Optional[str] = None
           note_after: Optional[str] = None
           text_before: Optional[str] = None  # for tag-type cells
           text_after: Optional[str] = None

       class MatrixCellUpdate(BaseModel):
           """PATCH-style update for a single matrix cell."""
           status: Optional[Literal[
               'completed', 'partial', 'in_progress',
               'not_done', 'not_relevant', 'no_findings',
           ]] = None
           note: Optional[str] = Field(None, max_length=500)
           text_value: Optional[str] = Field(None, max_length=200)  # for tag-type stages

       class MatrixCell(BaseModel):
           """Cell in the matrix — one (unit_id × stage_id) entry."""
           project_id: str
           unit_id: str  # use unit_id; for non-unit rows in PHASE 2, special location IDs
           stage_id: str
           status: Optional[str] = None
           text_value: Optional[str] = None
           note: Optional[str] = None
           audit: List[MatrixCellAuditEntry] = Field(default_factory=list)
           last_updated_at: Optional[str] = None
           last_updated_by: Optional[str] = None

       """
       Execution Matrix routes — Phase 1 backend.
       Implements the project-level matrix grid (units × stages) with:
         - Stage list customization (base from qc_template + custom additions)
         - Per-cell status + note + audit history
         - Per-user saved views

       NO modification of qc_template, qc_runs, or qc_items.
       READS qc_template stages live (computed render).
       READS qc_runs.stage_statuses for annotations (PHASE 2 only — not in PHASE 1).
       """
       from __future__ import annotations

       import uuid
       from typing import Optional, List, Dict, Any
       from fastapi import APIRouter, Depends, HTTPException, Query

       from contractor_ops.router import (
           _check_project_access, _now, _audit, get_db, require_roles,
       )
       from contractor_ops.qc_router import (
           _get_template,
           _get_project_role,
           _is_super_admin,
           MANAGEMENT_ROLES,
       )
       from contractor_ops.schemas import (
           MATRIX_STATUS_VALUES, MATRIX_STAGE_TYPES,
           MatrixStage, MatrixStageCreate, MatrixStagesUpdate,
           MatrixCell, MatrixCellUpdate, MatrixCellAuditEntry,
           MatrixSavedView, MatrixSavedViewCreate, MatrixSavedViewFilters,
       )

       router = APIRouter(prefix="/api/execution-matrix", tags=["execution-matrix"])

       # =====================================================================
       # RBAC helpers
       # =====================================================================

       async def _get_matrix_approver_user_ids(db, project_id: str) -> set:
           """Return set of user_ids that are 'בקרת ביצוע approvers' for this
           project. Uses the existing project_approvers collection (verified
           in STEP 0.2)."""
           # NOTE: actual collection name + field shape verified in STEP 0.2.
           # Likely query:
           cursor = db.project_approvers.find(
               {"project_id": project_id},
               {"_id": 0, "user_id": 1}
           )
           return {a["user_id"] async for a in cursor}

       async def _check_matrix_view(user, project_id):
           """Anyone in management_team / project_manager / super_admin can view.
           Contractors are explicitly blocked."""
           role = await _get_project_role(user, project_id)
           if _is_super_admin(user):
               return role
           if role == "contractor":
               raise HTTPException(status_code=403, detail="קבלן אינו רשאי לצפות במטריצת ביצוע")
           if role not in (*MANAGEMENT_ROLES, "project_manager", "viewer"):
               # viewer has read-only access to most project data; allow.
               # If role is unknown / unauthorized, block.
               raise HTTPException(status_code=403, detail="אין הרשאה לצפות במטריצת ביצוע")
           return role

       async def _check_matrix_edit(user, project_id):
           """Edit allowed for: project_manager, super_admin, OR anyone in
           the project's approvers list."""
           if _is_super_admin(user):
               return "super_admin"
           role = await _get_project_role(user, project_id)
           if role == "project_manager":
               return role
           db = get_db()
           approver_ids = await _get_matrix_approver_user_ids(db, project_id)
           if user["id"] in approver_ids:
               return "approver"
           raise HTTPException(
               status_code=403,
               detail="עריכה במטריצת ביצוע מותרת רק למנהל פרויקט או מאשרי בקרת ביצוע",
           )

       # =====================================================================
       # Lazy init
       # =====================================================================

       async def _get_or_init_matrix_config(db, project_id: str) -> dict:
           """Get the project's matrix config doc, or create empty one.
           Doc shape:
             {project_id, custom_stages: [], base_stages_removed: [],
              created_at, created_by, updated_at, updated_by}"""
           existing = await db.execution_matrix.find_one(
               {"project_id": project_id, "deletedAt": None},
               {"_id": 0}
           )
           if existing:
               return existing
           now = _now()
           doc = {
               "id": str(uuid.uuid4()),
               "project_id": project_id,
               "custom_stages": [],
               "base_stages_removed": [],
               "created_at": now,
               "created_by": None,
               "updated_at": now,
               "updated_by": None,
               "deletedAt": None,
           }
           await db.execution_matrix.insert_one(doc)
           return doc

       def _resolve_visible_stages(matrix_config: dict, tpl: dict) -> List[dict]:
           """Compute the visible stage list:
              base = qc_template.stages (filtered to remove base_stages_removed)
              custom = matrix_config.custom_stages
              merged = sort by .order
           """
           base_removed = set(matrix_config.get("base_stages_removed", []))
           base_stages = []
           for s in tpl.get("stages", []):
               if s["id"] in base_removed:
                   continue
               base_stages.append({
                   "id": s["id"],
                   "title": s.get("title", ""),
                   "type": "status",  # base stages are always status type
                   "order": s.get("order", 0),
                   "source": "base",
                   "scope": s.get("scope"),
               })
           custom_stages = []
           for cs in matrix_config.get("custom_stages", []):
               custom_stages.append({
                   "id": cs["id"],
                   "title": cs.get("title", ""),
                   "type": cs.get("type", "status"),
                   "order": cs.get("order", 9999),
                   "source": "custom",
                   "scope": None,  # custom stages aren't scoped
               })
           merged = sorted(base_stages + custom_stages, key=lambda x: x["order"])
           return merged

       # ENDPOINTS
       # =====================================================================

       @router.get("/{project_id}")
       async def get_matrix(
           project_id: str,
           user: dict = Depends(get_current_user)
       ):
           """Return the full matrix view for a project:
              - List of resolved stages (base + custom)
              - List of units (sorted by floor → unit_no)
              - List of cells (status + note + last_updated_by)
              - User's permissions (can_view, can_edit)
           """
           db = get_db()
           role = await _check_matrix_view(user, project_id)
           can_edit = False
           try:
               await _check_matrix_edit(user, project_id)
               can_edit = True
           except HTTPException:
               pass  # view-only

           matrix_config = await _get_or_init_matrix_config(db, project_id)
           tpl = await _get_template(db, project_id=project_id)
           stages = _resolve_visible_stages(matrix_config, tpl)

           # Units (sorted by floor + unit_no)
           # NOTE: requires JOIN with floors+buildings to sort properly
           units_cursor = db.units.find(
               {"project_id": project_id, "archived": {"$ne": True}},
               {"_id": 0}
           )
           units = await units_cursor.to_list(2000)
           # Enrich with floor + building data
           floor_ids = list({u["floor_id"] for u in units if "floor_id" in u})
           floors = await db.floors.find(
               {"id": {"$in": floor_ids}, "archived": {"$ne": True}},
               {"_id": 0}
           ).to_list(500)
           floors_by_id = {f["id"]: f for f in floors}
           # Sort units: by floor_number (floors_by_id[u.floor_id].floor_number), then unit_no
           def _unit_sort_key(u):
               f = floors_by_id.get(u.get("floor_id"), {})
               return (f.get("floor_number", 0), u.get("unit_no", ""))
           units.sort(key=_unit_sort_key)

           # Cells — fetch all for this project in one query
           stage_ids = [s["id"] for s in stages]
           cells_docs = await db.execution_matrix_cells.find(
               {"project_id": project_id, "stage_id": {"$in": stage_ids}},
               {"_id": 0}
           ).to_list(50000)

           # Strip audit history for list view (return only last entry summary)
           def _summarize_cell(c):
               last_audit = c.get("audit", [])[-1] if c.get("audit") else None
               return {
                   "unit_id": c["unit_id"],
                   "stage_id": c["stage_id"],
                   "status": c.get("status"),
                   "text_value": c.get("text_value"),
                   "note": c.get("note"),
                   "last_updated_at": c.get("last_updated_at"),
                   "last_updated_by": c.get("last_updated_by"),
                   "last_actor_name": last_audit.get("actor_name") if last_audit else None,
               }
           cells_summary = [_summarize_cell(c) for c in cells_docs]

           return {
               "project_id": project_id,
               "stages": stages,
               "units": units,
               "cells": cells_summary,
               "permissions": {
                   "role": role,
                   "can_view": True,
                   "can_edit": can_edit,
               },
               "matrix_config_id": matrix_config["id"],
           }

       @router.patch("/{project_id}/stages")
       async def update_stages(
           project_id: str,
           payload: MatrixStagesUpdate,
           user: dict = Depends(get_current_user)
       ):
           """Edit the project's stage list:
              - Add/remove/reorder custom stages
              - Hide/show base stages (toggle base_stages_removed)
           Allowed for: PM + super_admin + matrix approvers.
           """
           db = get_db()
           await _check_matrix_edit(user, project_id)

           matrix_config = await _get_or_init_matrix_config(db, project_id)
           updates = payload.dict(exclude_unset=True)

           # Process custom_stages_added: assign IDs + order
           if "custom_stages_added" in updates:
               max_existing_order = max(
                   (s.get("order", 0) for s in matrix_config.get("custom_stages", [])),
                   default=999,
               )
               new_custom = []
               for i, s in enumerate(updates["custom_stages_added"]):
                   new_custom.append({
                       "id": s.get("id") or f"custom_{uuid.uuid4().hex[:12]}",
                       "title": s["title"].strip(),
                       "type": s.get("type", "status"),
                       "order": s.get("order") if s.get("order") is not None else max_existing_order + i + 1,
                       "created_by": user["id"],
                       "created_at": _now(),
                   })
               updates["custom_stages"] = new_custom
               del updates["custom_stages_added"]

           if "base_stages_removed" in updates:
               # Sanity: stage_ids must be strings
               updates["base_stages_removed"] = [str(s) for s in updates["base_stages_removed"]]

           updates["updated_at"] = _now()
           updates["updated_by"] = user["id"]

           await db.execution_matrix.update_one(
               {"project_id": project_id, "deletedAt": None},
               {"$set": updates}
           )

           await _audit(
               "execution_matrix", project_id, "matrix_stages_updated",
               user["id"], {"updates": list(updates.keys())},
           )

           refreshed = await _get_or_init_matrix_config(db, project_id)
           tpl = await _get_template(db, project_id=project_id)
           return {
               "project_id": project_id,
               "stages": _resolve_visible_stages(refreshed, tpl),
           }

       @router.put("/{project_id}/cells/{unit_id}/{stage_id}")
       async def update_cell(
           project_id: str,
           unit_id: str,
           stage_id: str,
           payload: MatrixCellUpdate,
           user: dict = Depends(get_current_user)
       ):
           """Update a single matrix cell. Append audit entry.
           Allowed for: PM + super_admin + matrix approvers."""
           db = get_db()
           await _check_matrix_edit(user, project_id)

           # Verify stage exists in current visible stages (sanity)
           matrix_config = await _get_or_init_matrix_config(db, project_id)
           tpl = await _get_template(db, project_id=project_id)
           visible_stage_ids = {s["id"] for s in _resolve_visible_stages(matrix_config, tpl)}
           if stage_id not in visible_stage_ids:
               raise HTTPException(status_code=404, detail="שלב לא קיים במטריצה")

           # Verify unit_id exists in this project (or is a known non-unit location)
           unit = await db.units.find_one({"id": unit_id, "project_id": project_id})
           if not unit:
               raise HTTPException(status_code=404, detail="דירה לא נמצאה בפרויקט")

           # Status validation
           new_status = payload.status
           if new_status is not None and new_status not in MATRIX_STATUS_VALUES:
               raise HTTPException(status_code=400, detail=f"Status not in {MATRIX_STATUS_VALUES}")

           # Stage type vs payload validation — defensive (prevents engineers
           # from mixing status + tag inputs and corrupting cell semantics).
           # Find the stage's type from resolved stages.
           stage = next(
               (s for s in _resolve_visible_stages(matrix_config, tpl) if s["id"] == stage_id),
               None,
           )
           stage_type = (stage or {}).get("type", "status")
           if stage_type == "status" and payload.text_value not in (None, ""):
               raise HTTPException(
                   status_code=400,
                   detail="שלב מסוג סטטוס לא מקבל ערך טקסט (text_value)",
               )
           if stage_type == "tag" and payload.status is not None:
               raise HTTPException(
                   status_code=400,
                   detail="שלב מסוג תגית לא מקבל ערך סטטוס (status)",
               )

           existing = await db.execution_matrix_cells.find_one(
               {"project_id": project_id, "unit_id": unit_id, "stage_id": stage_id}
           )
           now = _now()
           actor_name = user.get("name") or user.get("email") or user["id"]

           audit_entry = {
               "actor_id": user["id"],
               "actor_name": actor_name,
               "timestamp": now,
               "status_before": existing.get("status") if existing else None,
               "status_after": new_status,
               "note_before": existing.get("note") if existing else None,
               "note_after": payload.note,
               "text_before": existing.get("text_value") if existing else None,
               "text_after": payload.text_value,
           }

           if existing:
               new_audit = existing.get("audit", []) + [audit_entry]
               await db.execution_matrix_cells.update_one(
                   {"project_id": project_id, "unit_id": unit_id, "stage_id": stage_id},
                   {"$set": {
                       "status": new_status,
                       "note": payload.note,
                       "text_value": payload.text_value,
                       "audit": new_audit,
                       "last_updated_at": now,
                       "last_updated_by": user["id"],
                   }}
               )
           else:
               await db.execution_matrix_cells.insert_one({
                   "id": str(uuid.uuid4()),
                   "project_id": project_id,
                   "unit_id": unit_id,
                   "stage_id": stage_id,
                   "status": new_status,
                   "note": payload.note,
                   "text_value": payload.text_value,
                   "audit": [audit_entry],
                   "last_updated_at": now,
                   "last_updated_by": user["id"],
                   "created_at": now,
               })

           return {
               "project_id": project_id,
               "unit_id": unit_id,
               "stage_id": stage_id,
               "status": new_status,
               "note": payload.note,
               "text_value": payload.text_value,
               "last_updated_at": now,
               "last_updated_by": user["id"],
               "last_actor_name": actor_name,
           }

       @router.get("/{project_id}/cells/{unit_id}/{stage_id}/history")

           )
           if result.matched_count == 0:
               raise HTTPException(status_code=404, detail="תצוגה לא נמצאה")
           return None
