CONTEXT
================================================================================

Discovered 2026-05-04 by Zahi during V9 visual smoke testing of Batch S2A
(#468 — Safety Project Registration). The form's Section 3 ("מנהלי החברה")
exposes only generic fields: first_name, last_name, id_number, address,
notes. There is no `role` / `title` field, and no way to mark someone as
מנהל עבודה vs עוזר בטיחות vs ממונה בטיחות.

This is a real gap because:

1. Israeli safety regulation REQUIRES distinct roles per project:
   - מנהל עבודה (Site Manager) — required at every site
   - עוזר בטיחות (Safety Assistant) — required at sites with 50+ workers
   - ממונה בטיחות (Safety Officer) — typically external, often shared
     across multiple projects of the same contractor
   - יועץ בטיחות (Safety Consultant) — optional advisor

2. Phase 2 Tour signatures will need to look up "the מנהל עבודה for this
   project" and "the עוזר בטיחות for this project" by role. Per concept
   decision #5 (HANDOFF doc), these two roles are required signers on
   every safety tour.

3. Section 3 stays as-is — it correctly captures *company directors*
   (regulatory requirement to list company managers in the registry).
   Personnel and directors are different concepts and should not be
   conflated.

DESIGN DECISION (Zahi 2026-05-04):
  - SEPARATE collection `safety_project_personnel` (NOT embedded in
    safety_project_settings). Reason: Phase 2 Tour signatures need stable
    UUID references to each person. Embedded list = reference by index
    (fragile) or by name (very fragile).
  - NO LIMIT on count per role. A project can have multiple מנהלי עבודה,
    multiple עוזרי בטיחות, etc.
  - Section in the form groups personnel by role with "+ הוסף [תפקיד]"
    button per role.

       """
       Safety Project Personnel routes — Batch S2B 2026-05-04.

       Distinct from SafetyProjectRegistration.managers (which captures
       company directors). This module manages PROJECT-LEVEL safety roles:
       מנהל עבודה / עוזר בטיחות / ממונה בטיחות / יועץ בטיחות.

       Stored in collection `safety_project_personnel`.
       Lazy init — collection grows on first POST. No migration needed.

       File created per CLAUDE.md rule: new sub-batches in new files.
       Imports helpers from contractor_ops.safety_router instead of
       duplicating them.
       """
       from __future__ import annotations

       import uuid
       from typing import Optional, List

       from fastapi import APIRouter, Depends, HTTPException, Query

       from contractor_ops.safety_router import (
           _check_project_access,
           _hash_id_number,
           _now_iso,
           SAFETY_VIEWERS,
           SAFETY_WRITERS,
           get_db,
       )
       from contractor_ops.router import require_roles
       from contractor_ops.schemas import (
           SafetyProjectPersonnel,
           SafetyProjectPersonnelCreate,
           SafetyProjectPersonnelUpdate,
       )

       # Same prefix as parent safety router — FastAPI merges them transparently.
       router = APIRouter(prefix="/api/safety", tags=["safety"])


       def _mask_id(raw: Optional[str]) -> Optional[str]:
           """Mask Israeli ID for UI display: '123456789' → '1***6789'.
           Same pattern used inline in safety_registration_router for
           SafetyProjectRegistration.managers."""
           if not raw:
               return None
           stripped = raw.strip()
           if len(stripped) < 4:
               return None  # Too short — don't display partial
           return f"{stripped[:1]}***{stripped[-3:]}"


       def _to_response(doc: dict) -> dict:
           """Strip Mongo _id + the hashed PII; return masked form."""
           out = {k: v for k, v in doc.items() if k not in ("_id", "id_number_hash", "deletedAt")}
           # id_number stored as masked already by create/update; pass through.
           return out


       @router.get("/{project_id}/personnel", response_model=List[SafetyProjectPersonnel])
       async def list_personnel(
           project_id: str,
           role: Optional[str] = Query(None, description="Filter by role"),
           is_active: Optional[bool] = Query(True, description="Default: only active"),
           user: dict = Depends(require_roles(*SAFETY_VIEWERS)),
       ):
           db = get_db()
           await _check_project_access(user, project_id)
           query: dict = {"project_id": project_id, "deletedAt": None}
           if role is not None:
               query["role"] = role
           if is_active is not None:
               query["is_active"] = is_active
           cursor = db.safety_project_personnel.find(query).sort([("role", 1), ("last_name", 1), ("first_name", 1)])
           docs = await cursor.to_list(length=500)
           return [_to_response(d) for d in docs]


       @router.post("/{project_id}/personnel", response_model=SafetyProjectPersonnel, status_code=201)
       async def create_personnel(
           project_id: str,
           payload: SafetyProjectPersonnelCreate,
           user: dict = Depends(require_roles(*SAFETY_WRITERS)),
       ):
           db = get_db()
           await _check_project_access(user, project_id)
           now = _now_iso()
           raw_id = payload.id_number
           doc = {
               "id": str(uuid.uuid4()),
               "project_id": project_id,
               "role": payload.role,
               "first_name": payload.first_name.strip(),
               "last_name": payload.last_name.strip(),
               "id_number": _mask_id(raw_id),  # masked for display
               "id_number_hash": _hash_id_number(raw_id) if raw_id else None,  # PII protection
               "license_number": payload.license_number,
               "phone": payload.phone,
               "email": payload.email,
               "notes": payload.notes,
               "is_active": True,
               "created_at": now,
               "created_by": user.get("email") or user.get("id"),
               "updated_at": now,
               "updated_by": user.get("email") or user.get("id"),
               "deletedAt": None,
           }
           await db.safety_project_personnel.insert_one(doc)
           return _to_response(doc)


       @router.patch("/{project_id}/personnel/{personnel_id}", response_model=SafetyProjectPersonnel)
       async def update_personnel(
           project_id: str,
           personnel_id: str,
           payload: SafetyProjectPersonnelUpdate,
           user: dict = Depends(require_roles(*SAFETY_WRITERS)),
       ):
           db = get_db()
           await _check_project_access(user, project_id)
           existing = await db.safety_project_personnel.find_one(
               {"id": personnel_id, "project_id": project_id, "deletedAt": None}
           )
           if not existing:
               raise HTTPException(status_code=404, detail="Personnel not found")

           updates = payload.dict(exclude_unset=True)

           # Handle id_number specially: re-hash + re-mask, never persist raw.
           if "id_number" in updates:
               raw = updates["id_number"]
               updates["id_number"] = _mask_id(raw)
               updates["id_number_hash"] = _hash_id_number(raw) if raw else None

           # Trim string fields where present.
           for k in ("first_name", "last_name"):
               if k in updates and isinstance(updates[k], str):
                   updates[k] = updates[k].strip()

           updates["updated_at"] = _now_iso()
           updates["updated_by"] = user.get("email") or user.get("id")

           await db.safety_project_personnel.update_one(
               {"id": personnel_id, "project_id": project_id, "deletedAt": None},
               {"$set": updates},
           )
           refreshed = await db.safety_project_personnel.find_one(
               {"id": personnel_id, "project_id": project_id}
           )
           return _to_response(refreshed)


       @router.delete("/{project_id}/personnel/{personnel_id}", status_code=204)
       async def delete_personnel(
           project_id: str,
           personnel_id: str,
           user: dict = Depends(require_roles(*SAFETY_WRITERS)),
       ):
           db = get_db()
           await _check_project_access(user, project_id)
           result = await db.safety_project_personnel.update_one(
               {"id": personnel_id, "project_id": project_id, "deletedAt": None},
               {
                   "$set": {
                       "deletedAt": _now_iso(),
                       "is_active": False,
                       "updated_at": _now_iso(),
                       "updated_by": user.get("email") or user.get("id"),
                   }
               },
           )
           if result.matched_count == 0:
               raise HTTPException(status_code=404, detail="Personnel not found")
           return None

NOTES:

  in safety_router.py for workers).
- Same prefix `/api/safety` as parent — FastAPI merges. So actual paths:

    POST   /api/safety/{project_id}/personnel
    PATCH  /api/safety/{project_id}/personnel/{personnel_id}
    DELETE /api/safety/{project_id}/personnel/{personnel_id}
- If `_hash_id_number` or `_check_project_access` import path differs in the
  current safety_router.py (e.g. `_check_project_access` is in router.py
  instead) — adapt the import. STEP 0.1 verifies actual location.

       /**
        * Safety Project Personnel Section — Batch S2B 2026-05-04.
        *
        * Renders inside SafetyProjectRegistrationPage as a collapsible section.
        * Lists personnel grouped by 4 safety roles. Each role group has
        * "+ הוסף" button. Multiple personnel per role allowed.
        *
        * PII: ID number is sent raw on POST/PATCH but server returns masked
        * form ("1***6789"). The masked form is what we display + persist
        * locally. Never display raw ID after creation.
        *
        * Capacitor: all touch targets py-3 (≥44px). No file downloads in

        */
       import React, { useEffect, useState, useCallback } from 'react';
       import { Plus, Trash2, ChevronDown, ChevronUp } from 'lucide-react';
       import { safetyService } from '../../services/api';

       const ROLE_LABELS = {
         site_manager:      'מנהל עבודה',
         safety_assistant:  'עוזר בטיחות',
         safety_officer:    'ממונה בטיחות',
         safety_consultant: 'יועץ בטיחות',
       };

       const ROLE_ORDER = ['site_manager', 'safety_assistant', 'safety_officer', 'safety_consultant'];

       const EMPTY_DRAFT = {
         first_name: '',
         last_name: '',
         id_number: '',
         license_number: '',
         phone: '',
         email: '',
         notes: '',
       };

       export default function SafetyProjectPersonnelSection({ projectId, isOpen, onToggle }) {
         const [personnel, setPersonnel] = useState([]);
         const [loading, setLoading] = useState(true);
         const [error, setError] = useState(null);
         const [drafts, setDrafts] = useState({}); // { role: { ...EMPTY_DRAFT } } for in-progress add
         const [savingRole, setSavingRole] = useState(null);

         const reload = useCallback(async () => {
           try {
             setLoading(true);
             const data = await safetyService.listPersonnel(projectId, { is_active: true });
             setPersonnel(Array.isArray(data) ? data : []);
             setError(null);
           } catch (err) {
             console.error('listPersonnel failed:', err);
             setError('טעינה נכשלה');
           } finally {
             setLoading(false);
           }
         }, [projectId]);

         useEffect(() => {
           if (projectId) reload();
         }, [projectId, reload]);

         const startAdd = (role) => {
           setDrafts(prev => ({ ...prev, [role]: { ...EMPTY_DRAFT } }));
         };

         const cancelAdd = (role) => {
           setDrafts(prev => {
             const next = { ...prev };
             delete next[role];
             return next;
           });
         };

         const updateDraft = (role, field, value) => {
           setDrafts(prev => ({ ...prev, [role]: { ...prev[role], [field]: value } }));
         };

         const submitDraft = async (role) => {
           const draft = drafts[role];
           if (!draft || !draft.first_name.trim() || !draft.last_name.trim()) {
             setError('שם פרטי + שם משפחה הם שדות חובה');
             return;
           }
           try {
             setSavingRole(role);
             const payload = {
               role,
               first_name: draft.first_name.trim(),
               last_name: draft.last_name.trim(),
               id_number: draft.id_number?.trim() || null,
               license_number: draft.license_number?.trim() || null,
               phone: draft.phone?.trim() || null,
               email: draft.email?.trim() || null,
               notes: draft.notes?.trim() || null,
             };
             await safetyService.createPersonnel(projectId, payload);
             cancelAdd(role);
             await reload();
           } catch (err) {
             console.error('createPersonnel failed:', err);
             setError('שמירה נכשלה');
           } finally {
             setSavingRole(null);
           }
         };

         const remove = async (personnelId) => {
           if (!window.confirm('להסיר את האחראי הזה?')) return;
           try {
             await safetyService.deletePersonnel(projectId, personnelId);
             await reload();
           } catch (err) {
             console.error('deletePersonnel failed:', err);
             setError('מחיקה נכשלה');
           }
         };

         const grouped = ROLE_ORDER.reduce((acc, role) => {
           acc[role] = personnel.filter(p => p.role === role);
           return acc;
         }, {});

         const totalCount = personnel.length;

         return (
           <div className="border border-slate-200 rounded-lg bg-white">
             {/* Section header — collapsible */}
             <button
               type="button"
               onClick={onToggle}
               className="w-full flex items-center justify-between py-3 px-4 text-right"
             >
               <span className="text-slate-500">
                 {isOpen ? <ChevronUp className="w-5 h-5" /> : <ChevronDown className="w-5 h-5" />}
               </span>
               <span className="font-semibold text-slate-800">
                 4. אחראי בטיחות בפרויקט ({totalCount})
               </span>
             </button>

             {isOpen && (
               <div className="px-4 pb-4 space-y-4">
                 {error && (
                   <div className="text-sm text-red-600 bg-red-50 border border-red-200 rounded p-2">
                     {error}
                   </div>
                 )}

                 {loading ? (
                   <p className="text-sm text-slate-500">טוען...</p>
                 ) : (
                   ROLE_ORDER.map(role => (
                     <div key={role} className="border-t border-slate-100 pt-3 first:border-t-0 first:pt-0">
                       <div className="flex items-center justify-between mb-2">
                         <span className="text-sm text-slate-500">
                           {grouped[role].length} {grouped[role].length === 1 ? 'איש' : 'אנשים'}
                         </span>
                         <h4 className="font-medium text-slate-700">{ROLE_LABELS[role]}</h4>
                       </div>

                       {/* Existing personnel list for this role */}
                       {grouped[role].length === 0 && !drafts[role] && (
                         <p className="text-xs text-slate-400 mb-2">לא הוגדרו {ROLE_LABELS[role]}</p>
                       )}
                       <div className="space-y-2 mb-2">
                         {grouped[role].map(p => (

                             <button
                               type="button"
                               onClick={() => remove(p.id)}
                               className="text-red-500 hover:text-red-700 p-2"
                               aria-label="הסר"
                             >
                               <Trash2 className="w-4 h-4" />
                             </button>
                             <div className="flex-1 text-right space-y-1">
                               <div className="font-medium text-slate-800">

                               </div>

                                 <div className="text-xs text-slate-500">ת.ז: {p.id_number}</div>
                               )}
                               {p.license_number && (
                                 <div className="text-xs text-slate-500">רישיון: {p.license_number}</div>
                               )}
                               {p.phone && (
                                 <div className="text-xs text-slate-500">טל': {p.phone}</div>
                               )}
                               {p.email && (
                                 <div className="text-xs text-slate-500">דוא״ל: {p.email}</div>
                               )}

                                 <div className="text-xs text-slate-500 mt-1">{p.notes}</div>
                               )}
                             </div>
                           </div>
                         ))}
                       </div>

                       {/* Inline add draft for this role */}
                       {drafts[role] ? (
                         <div className="border border-blue-200 rounded p-3 bg-blue-50 space-y-2">
                           <div className="grid grid-cols-2 gap-2">
                             <input
                               type="text"
                               placeholder="שם פרטי *"
                               value={drafts[role].first_name}
                               onChange={(e) => updateDraft(role, 'first_name', e.target.value)}
                               className="border border-slate-300 rounded px-3 py-3 text-right text-sm"
                             />
                             <input
                               type="text"
                               placeholder="שם משפחה *"
                               value={drafts[role].last_name}
                               onChange={(e) => updateDraft(role, 'last_name', e.target.value)}
                               className="border border-slate-300 rounded px-3 py-3 text-right text-sm"
                             />
                           </div>
                           <input
                             type="text"
                             placeholder="ת.ז (אופציונלי)"
                             value={drafts[role].id_number}

                             className="w-full border border-slate-300 rounded px-3 py-3 text-right text-sm"

                             />

                               placeholder="דוא״ל"
                               value={drafts[role].email}
                               onChange={(e) => updateDraft(role, 'email', e.target.value)}
                               className="border border-slate-300 rounded px-3 py-3 text-right text-sm"
                             />
                           </div>
                           <textarea
                             placeholder="הערות"
                             value={drafts[role].notes}
                             onChange={(e) => updateDraft(role, 'notes', e.target.value)}
                             rows={2}
                             className="w-full border border-slate-300 rounded px-3 py-3 text-right text-sm"
                           />
                           <div className="flex items-center gap-2 justify-end">
                             <button
                               type="button"
                               onClick={() => cancelAdd(role)}
                               className="px-4 py-3 text-sm text-slate-600 hover:bg-slate-100 rounded"
                               disabled={savingRole === role}
                             >
                               ביטול
                             </button>
                             <button
                               type="button"
                               onClick={() => submitDraft(role)}
                               className="px-4 py-3 text-sm bg-blue-600 text-white rounded hover:bg-blue-700 disabled:opacity-50"
                               disabled={savingRole === role}
                             >
                               {savingRole === role ? 'שומר...' : 'שמור'}
                             </button>
                           </div>
                         </div>
                       ) : (
                         <button
                           type="button"
                           onClick={() => startAdd(role)}
                           className="w-full flex items-center justify-center gap-2 py-3 px-4 border border-dashed border-slate-300 rounded text-sm text-slate-600 hover:bg-slate-50"
                         >
                           <Plus className="w-4 h-4" /> הוסף {ROLE_LABELS[role]}
                         </button>
                       )}
                     </div>
                   ))
                 )}
               </div>
             )}
           </div>
         );
       }

NOTES:
